## Introduction

A Kotlin I/O toolkit for _bigdata_ formats.

Allows you to read and write [Parquet](https://parquet.apache.org/) and [Orc](https://orc.apache.org/) files in Kotlin.

Compatible with data generated by Apache Spark and does not require you to start a cluster to perform I/O operations.
